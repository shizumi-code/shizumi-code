### Short Description

> **IllustriousTagTool** is an offline Windows tool that uses a local LLM to organize text into tag classification candidates.
> No internet connection required. All data is managed safely via CSV files.

---

### Slightly Longer Description

> IllustriousTagTool is a lightweight desktop application for Windows that helps you organize text into tag classification candidates using a local large language model (LLM).
>
> It runs completely offline, does not send data to external servers, and stores all results in simple CSV files that you fully control.
>
> This project focuses on stability, transparency, and safe data handling rather than automated decision-making.
